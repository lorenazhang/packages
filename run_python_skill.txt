# Run Python code with miniforge_env

Always use ONLY this Python executable:

```powershell
& "C:\Users\u781896\AppData\Local\miniforge3_wf\envs\miniforge_env\python.exe"
```

NEVER use system Python.
NEVER use plain `python`.
NEVER fall back to system Python.

Steps:

1. First verify Python executable:

```powershell
& "C:\Users\u781896\AppData\Local\miniforge3_wf\envs\miniforge_env\python.exe" -c "import sys; print(sys.executable)"
```

2. Then run the file:

```powershell
& "C:\Users\u781896\AppData\Local\miniforge3_wf\envs\miniforge_env\python.exe" <file.py>
```

3. Return the Python executable path, stdout, and stderr.

4. If this exact Python executable path does not work, stop and report the error. Do not use any other Python.
